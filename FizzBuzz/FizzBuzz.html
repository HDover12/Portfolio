FizzBuzz Challenge

The Challenge

For this challenge from CodeAcademy.com, I created a fizzbuzz() function that takes in a number, n, and returns an array of the numbers from 1 to n. For multiples of three, the array returns "Fizz" instead of the number, and for the multiples of five, "Buzz". For numbers that are multiples of both three and five, "FizzBuzz" will be returned. For example, fizzbuzz(16) should return [1, 2, 'Fizz', 4, 'Buzz', 'Fizz', 7, 8, 'Fizz', 'Buzz', 11, 'Fizz', 13, 14, 'FizzBuzz', 16].


My Solution

My approach to this challenge was to loop through until each number up to "n" was added to the array checking along the way if that number was divisible by 3, 5, or both and replacing the number accordingly.
